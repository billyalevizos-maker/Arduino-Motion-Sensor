Arduino Motion Sensor with Servo and Buzzer

# Overview

This project uses an ultrasonic sensor to detect the distance of an object and respond with both motion and sound. As an object approaches, a servo motor rotates, and a buzzer activates when the object is within a 10cm of the ultrasonic sensor.

# Components

* Arduino Uno
* Ultrasonic distance sensor
* Servo motor
* Buzzer
* Jumper wires and breadboard

# How It Works

The ultrasonic sensor continuously measures the distance to an object.

* When an object moves closer, the Arduino processes the distance data
* The servo motor rotates proportionally
* When the distance reaches 10 cm or less, the buzzer is activated as an alert
